# Changelog

## v2.0.0 (2026-06-27)

**Spatial analysis unchanged; temporal framing corrected.**

The data, the pooled ⟨r⟩ = 0.344 / CV = 1.45 clustering result, the independent
corroboration, all figures, and the demonstration that declustering fabricates
GOE are exactly as in v1 and stand.

Changed:
- **Temporal leg reframed.** v1 described orogenic-gold mineralization timing as
  "temporal level repulsion (GOE)" and cast the paper as a "time–space
  asymmetry." Per a subsequent audit (Chen 2026, Zenodo
  10.5281/zenodo.20980670), a charge-and-release source is a relaxation
  oscillator whose timing is a narrow **scalar clock** — a high but
  shuffle-invariant spacing ratio with no spectral rigidity, and unresolvable
  given geochronological age errors — **not** level repulsion. Genuine
  Wigner–Dyson statistics require a chaotic, strongly-mixed spectrum.
- **Result restated** as "temporal scalar clock + spatial clustering" (not a
  time–space asymmetry).
- **Title** changed to headline the robust spatial result and the declustering
  warning.
- Added a "Changes from version 1 (erratum)" section and a citation to the audit.
- v1 PDF preserved at `paper/archive/v1/`.

The superposition→Poisson principle, on which the spatial argument rests, is
unaffected.
